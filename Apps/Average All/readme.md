# Hubitat

Average All 
**************************


INSTALL NOTES:
********************

1. Copy the Parent code from GitHub into a ‘New App’ under the ‘Apps Code’ menu then click ‘Save’

2. Copy the Child code from GitHub into a second ‘New App’ & save this too

3. Go to ‘Apps’ 

4. Click ‘Load New Apps’

5. Select ‘Average All’ under ‘User Apps’ (Not the child app)

6. Done

7. Go back to Apps and open Average All 

You can now create new child apps directly from here.










This app will average out a number of physical illuminance sensors, humidity sensors or temperature sensors to set the value on a virtual device


You need to install the relevant virtual device from the drivers directory
