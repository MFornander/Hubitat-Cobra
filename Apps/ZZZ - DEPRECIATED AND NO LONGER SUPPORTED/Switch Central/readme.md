# Switch Central


Files to install:

1. Switch Central - Parent.groovy
2. Switch Central - One To Many.groovy
3. Switch Central - Switch Changeover.groovy
4. Switch Central - Scheduled Switch.groovy




INSTALL NOTES:
******************

1. Copy the Parent code from GitHub into a ‘New App’ under the ‘Apps Code’ menu then click ‘Save’

2. Copy each of the child codes from GitHub into their own ‘New App’ & save these too

3. Go to ‘Apps’ 

4. Click ‘Load New Apps’

5. Select ‘Switch Central’ under ‘User Apps’ (Not any of the child apps)

6. Save

7. Go back to Apps and open Switch Central 

You can now create new child apps directly from within Switch Central.
